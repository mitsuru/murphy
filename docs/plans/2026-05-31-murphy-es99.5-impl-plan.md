# VarSemanticModel Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Implement `VarSemanticModel` — a shared variable scope/data flow analysis engine in `murphy-core` — and migrate/add the 5 cops that depend on it.

**Architecture:** Pre-computed Ruff-style query model built once per file (`VarSemanticModel::build(&ast)`) before the cop loop. Stored in `CxRaw` as `*const VarSemanticModel`. Cops access it via `cx.var_model()`. Branch-aware `is_referenced` is pre-computed for every `Assignment` by porting the existing branch chain logic from `useless_assignment.rs`.

**Tech Stack:** Rust, murphy-core, murphy-plugin-api (CxRaw/Cx), murphy-std (cops). `cargo test --workspace` for verification.

```rust
use std::collections::HashMap;
use murphy_ast::{Ast, NodeId, NodeKind, OptNodeId, Symbol};

pub struct VarSemanticModel {
    scopes: HashMap<NodeId, ScopeInfo>,
}

pub struct ScopeInfo {
    pub parent_scope: Option<NodeId>,
    pub variables: Vec<Variable>,
}

pub struct Variable {
    pub name: Symbol,
    pub is_argument: bool,
    pub declaration_node: NodeId,
    pub assignments: Vec<Assignment>,
    pub references: Vec<Reference>,
}

pub struct Assignment {
    pub node_id: NodeId,
    /// Byte position of the assignment end (for ordering vs reads).
    pub end: u32,
    /// Pre-computed branch-aware: true iff value is observed on all paths.
    pub is_referenced: bool,
}

pub struct Reference {
    pub node_id: NodeId,
    pub pos: u32,
}

impl VarSemanticModel {
    pub fn scope(&self, boundary_node: NodeId) -> Option<&ScopeInfo> {
        self.scopes.get(&boundary_node)
    }
}

impl ScopeInfo {
    pub fn parent_scope<'a>(&self, model: &'a VarSemanticModel) -> Option<&'a ScopeInfo> {
        self.parent_scope.and_then(|id| model.scope(id))
    }

    pub fn variables(&self) -> &[Variable] {
        &self.variables
    }
}
```

In `dfs_build`, when the DFS visits an `Args` node within a `Def`/`Defs`/`Block`, collect its parameters as `Variable { is_argument: true }`. Argument `NodeKind` variants:

### Step 4: Wire up Lvasgn / Lvar collection

In `dfs_build`:
- `NodeKind::Lvasgn { name, value }` where `value.is_some()` → push `Assignment` to current scope's variable
- `NodeKind::Lvar(name)` → push `Reference`
- `NodeKind::OpAsgn { target, .. }` / `OrAsgn` / `AndAsgn` → both a `Reference` and an `Assignment` on the `Lvasgn` target
- `NodeKind::Masgn { lhs, .. }` → walk `Mlhs` to collect `Lvasgn` targets
- `NodeKind::Resbody { var, .. }` → the `var` `Lvasgn` is an `Assignment`
- `NodeKind::For { var, .. }` → the `var` `Lvasgn` is an `Assignment`

Stop descending into scope boundaries (they get their own scope).

- `is_branch_barrier(ast, node) -> bool`
- `chain_is_prefix(short, long) -> bool`

### Step 1: Port helper functions

```rust
fn barrier_chain(ast: &Ast, root: NodeId, node: NodeId) -> Vec<(NodeId, NodeId)> {
    let mut chain: Vec<(NodeId, NodeId)> = Vec::new();
    let mut current = node;
    while let Some(parent) = ast.nodes()[current.0 as usize].parent.get() {
        if parent == root { break; }
        if is_branch_barrier(ast, parent) {
            chain.push((parent, current));
        }
        current = parent;
    }
    chain.reverse();
    chain
}

fn is_branch_barrier(ast: &Ast, node: NodeId) -> bool {
    matches!(
        *ast.kind(node),
        NodeKind::If { .. }
            | NodeKind::Case { .. }
            | NodeKind::When { .. }
            | NodeKind::While { .. }
            | NodeKind::Until { .. }
    )
}

fn chain_is_prefix(short: &[(NodeId, NodeId)], long: &[(NodeId, NodeId)]) -> bool {
    short.len() <= long.len() && short == &long[..short.len()]
}

fn paths_compatible(a: &[(NodeId, NodeId)], b: &[(NodeId, NodeId)]) -> bool {
    for (barrier_a, arm_a) in a {
        for (barrier_b, arm_b) in b {
            if barrier_a == barrier_b && arm_a != arm_b { return false; }
        }
    }
    true
}

fn is_in_protected_begin_body(ast: &Ast, root: NodeId, node: NodeId) -> bool {
    let mut current = node;
    while let Some(parent) = ast.nodes()[current.0 as usize].parent.get() {
        if parent == root { return false; }
        let body = match *ast.kind(parent) {
            NodeKind::Rescue { body, .. } | NodeKind::Ensure { body, .. } => body,
            _ => { current = parent; continue; }
        };
        if body.get() == Some(current) { return true; }
        current = parent;
    }
    false
}
```

### Step 2: Implement `analyze_scope_is_referenced`

Called on scope pop, sets `is_referenced` on every `Assignment` in the scope:

```rust
fn analyze_scope_is_referenced(ast: &Ast, scope_root: NodeId, scope: &mut ScopeInfo) {
    for var in &mut scope.variables {
        // Pre-compute branch chains for all assignments and references.
        let asgn_chains: Vec<_> = var.assignments.iter()
            .map(|a| barrier_chain(ast, scope_root, a.node_id))
            .collect();
        let ref_chains: Vec<_> = var.references.iter()
            .map(|r| barrier_chain(ast, scope_root, r.node_id))
            .collect();

        for (i, asgn) in var.assignments.iter_mut().enumerate() {
            // Loop body assignments are always considered referenced
            // (later iterations may observe them).
            if is_in_loop_body(ast, scope_root, asgn.node_id) {
                asgn.is_referenced = true;
                continue;
            }

            let next_read_pos = var.references.iter().enumerate()
                .filter(|(_, r)| r.pos > asgn.end)
                .filter(|(k, _)| paths_compatible(&ref_chains[*k], &asgn_chains[i]))
                .map(|(_, r)| r.pos)
                .min();

            let dominating_overwrite = var.assignments.iter().enumerate()
                .filter(|(j, w)| *j != i && w.end > asgn.end)
                .filter(|(j, _)| chain_is_prefix(&asgn_chains[*j], &asgn_chains[i]))
                .filter(|(_, w)| !is_in_protected_begin_body(ast, scope_root, w.node_id))
                .min_by_key(|(_, w)| w.end);

            asgn.is_referenced = match (next_read_pos, dominating_overwrite) {
                (None, _) => false,
                (Some(r), Some((_, w))) if w.end <= r => false,
                (Some(_), _) => true,
            };
        }
    }
}

fn is_in_loop_body(ast: &Ast, root: NodeId, node: NodeId) -> bool {
    let mut current = node;
    while let Some(parent) = ast.nodes()[current.0 as usize].parent.get() {
        if parent == root { return false; }
        match *ast.kind(parent) {
            NodeKind::While { body, .. } | NodeKind::Until { body, .. }
                if body == current => return true,
            NodeKind::For { body, .. } if body == current => return true,
            _ => {}
        }
        current = parent;
    }
    false
}
```

```rust
#[cop(
    name = "Lint/ShadowingOuterLocalVariable",
    description = "Detect block arguments that shadow local variables from an outer scope.",
    default_severity = "warning",
    default_enabled = true,
    options = NoOptions
)]
impl ShadowingOuterLocalVariable {
    #[on_node(kind = "block")]
    fn check(&self, node: NodeId, cx: &Cx<'_>) {
        let Some(model) = cx.var_model() else { return };
        let Some(scope) = model.scope(node) else { return };
        for var in scope.variables().iter().filter(|v| v.is_argument) {
            let mut outer = scope.parent_scope(model);
            while let Some(s) = outer {
                if s.variables().iter().any(|v| v.name == var.name) {
                    let name_str = cx.symbol_str(var.name);
                    let range = cx.node(var.declaration_node).loc.name;
                    cx.emit_offense(
                        range,
                        &format!("Shadowing outer local variable - `{name_str}`."),
                        None,
                    );
                    break;
                }
                outer = s.parent_scope(model);
            }
        }
    }
}
```

```rust
    #[on_node(kind = "block")]
    fn check(&self, node: NodeId, cx: &Cx<'_>) {
        let Some(model) = cx.var_model() else { return };
        let Some(scope) = model.scope(node) else { return };
        for var in scope.variables().iter().filter(|v| v.is_argument) {
            let name_str = cx.symbol_str(var.name);
            if name_str.starts_with('_') { continue; }
            // Check if the variable is referenced.
            if !var.references.is_empty() { continue; }
            let range = cx.node(var.declaration_node).loc.name;
            cx.emit_offense(
                range,
                &format!("Unused block argument - `{name_str}`."),
                None,
            );
            // Autocorrect: prefix name with `_`.
            cx.emit_edit(
                Range { start: range.start, end: range.start },
                "_",
            );
        }
    }
}
```

```rust
#[cop(
    name = "Lint/UnderscorePrefixedVariableName",
    description = "Flag underscore-prefixed local variables that are actually referenced.",
    default_severity = "warning",
    default_enabled = true,
    options = NoOptions
)]
impl UnderscorePrefixedVariableName {
    #[on_new_investigation]
    fn check_file(&self, cx: &Cx<'_>) {
        let Some(model) = cx.var_model() else { return };
        for (_, scope) in model.scopes() {
            for var in scope.variables() {
                let name_str = cx.symbol_str(var.name);
                if !name_str.starts_with('_') || name_str == "_" { continue; }
                if var.references.is_empty() { continue; }
                // The variable is referenced despite the underscore prefix.
                if let Some(asgn) = var.assignments.first() {
                    let range = cx.node(asgn.node_id).loc.name;
                    cx.emit_offense(
                        range,
                        &format!("Do not use `{name_str}` as a local variable name."),
                        None,
                    );
                }
            }
        }
    }
}
```
